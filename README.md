# BrubeckScan

## Description

BrubeckScan is a dApp used by node runners of the Streamr mainnet to check nodes statuses and earnings.

## Setup local dev env

```shell
# Get the project locally
git clone git@github.com:adamphivo/brubeckscan.git

# Install dependencies
npm install

# Copy the .env file provided at the root of the project

# Run dev server
npm run dev

# Build for production
npm run build
```
